synthetic_humanoid.vrm

A synthetic 20-joint VRM 1.0 humanoid skeleton authored programmatically
for the PSO Editor's import test suite. There is no mesh, texture, or
proprietary data — only the joint hierarchy + the standard VRM humanoid
``humanBones`` map. The file exists to exercise the VRM-detection +
retarget-routing path without requiring users to download a third-party
character.

Real CC0 VRM characters
-----------------------
For end-to-end visual smoke tests with actual mesh + texture data,
download a CC0-licensed character from any of:

  * VRoid Hub        https://hub.vroid.com/      (filter by license)
  * Booth-PM         https://booth.pm/           (search "VRM CC0")
  * Sketchfab        https://sketchfab.com/      (filter "Downloadable + CC0")

Drop the .vrm file into this directory; the
``test_local_cc0_vrm_smoke`` test will pick it up automatically.
